# 2.创建对象的过程

```js
function Monther(lastName){
  this.lastName=lastName
}
let son=new Mother("da")
```

```xml
1.创建了新对象 son.__proto__会和Mother.prototype联系起来
2.新对象和函数会用Mother.call(son,'da')进行绑定
3.执行代码
4.如果mother里边没有返回值，则返回创建的对象实例
```
